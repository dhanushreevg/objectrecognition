# Object Recognition Using Pre-trained Model

This project uses a pre-trained deep learning model to detect objects in images or video using Python.

## Requirements
- Python 3.x
- OpenCV
- NumPy

## How to Run
1. Install required libraries:
pip install opencv-python numpy


2. Run the program:
python main.py


## Output
The program detects objects and displays bounding boxes on the screen.
